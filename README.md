# سنة سترايد — SunnahStride v0.3 🕌

**حلال في كل لقمة • سنة في كل خطوة**

تطبيق Flutter متكامل للتغذية الحلال واللياقة الإسلامية والصحة الشاملة.

---

## 🚀 تشغيل التطبيق (Quick Start)

### المتطلبات
- Flutter SDK 3.x+ → https://flutter.dev/docs/get-started/install
- Dart 3.x+
- Android Studio أو VS Code

### خطوات التشغيل

```bash
# 1. تثبيت الاعتماديات
flutter pub get

# 2. التشغيل على الجهاز المتصل أو المحاكي
flutter run

# 3. بناء APK للأندرويد
flutter build apk --release

# 4. بناء للـ iOS (يحتاج macOS)
flutter build ios --release
```

## ✨ المميزات

### 🌙 Dark Mode
- وضع ليلي كامل بألوان مريحة للعيون
- زر تبديل في شريط التطبيق وصفحة الملف الشخصي
- يُحفظ الاختيار تلقائياً

### 📷 ماسح الحلال
- قاعدة بيانات مدمجة (يمكن استبدالها بـ API حقيقية)
- نتائج: حلال ✅ / مشبوه ⚠️ / حرام ❌ / غير معروف ❓
- سجل الماسحات

### 🔥 عداد السعرات
- حلقة SVG دائرية تتغير بالألوان
- إضافة وجبات سريعة (أطعمة سنية جاهزة)
- إدخال يدوي مخصص
- تتبع المغذيات (كربوهيدرات، بروتين، دهون)
- الإضافة المباشرة من الوصفات

### 🩺 الصحة والعافية
**تتبع:**
- 💧 الماء اليومي (أكواب + تقدم مرئي)
- 😴 النوم (ساعات + جودة)
- 🚶 الخطوات اليومية
- 😊 تتبع المزاج
- ❤️ معدل ضربات القلب

**حاسبات:**
- ⚖️ BMI مع مقياس مرئي ونصائح
- 🏃 السعرات المحروقة
- 💧 احتياج الماء المثالي

**مقالات صحية (10 مقالات):**
- الماء • النوم • القلب • الدماغ
- العظام • الأمعاء • الصحة النفسية
- الوزن • المكملات • الفحوصات الدورية

### 🏃 اللياقة السنية
- 10 تمارين مصنفة (إخوة / أخوات / الجميع)
- مؤقت دائري مع الحديث النبوي
- وضع أخوات (محتشم دائماً)

## ⚙️ إضافات مستقبلية

### الأولوية العالية
1. **RevenueCat** لنظام الاشتراك الحقيقي
2. **Firebase Auth** لتسجيل الدخول
3. **Aladhan API** لمواقيت الصلاة الحقيقية
4. **قاعدة بيانات حلال** من Dar al-Ifta / IFANCA

### الأولوية المتوسطة
5. **Push Notifications** لتذكير الصلاة والماء
6. **Claude API** لمخطط الوجبات الحقيقي
7. **camera + ML Kit** للماسح الضوئي الحقيقي
8. **مشاركة عائلية** (حتى 4 أشخاص)
